ІНСТРУКЦІЯ ДЛЯ ЗАПУСКУ ВИПРАВЛЕНЬ БАЗИ ДАНИХ

Через мережеві обмеження (відсутність прямого доступу до бази даних через IPv4 та обмеження RPC функцій для DDL операцій), автоматичне виконання SQL скриптів неможливе.

Будь ласка, виконайте наступні кроки вручну:

1. Відкрийте Supabase Dashboard вашого проекту: https://supabase.com/dashboard/project/aadrklaiemgilkaflvcy
2. Перейдіть у розділ "SQL Editor" (зліва в меню).
3. Натисніть "New query".
4. Скопіюйте вміст файлу `manual_fixes_all.sql` (знаходиться в корені проекту `d:\Avtoservis 2.0\manual_fixes_all.sql`).
5. Вставте скопійований код у вікно редактора SQL.
6. Натисніть кнопку "Run" (зелена кнопка).

Це виправить:
- Відсутні колонки `date`, `time`, `status` у таблиці `appointments`.
- Створить необхідні індекси.
- Додасть тестові дані та виправить зв'язки між записами.
