## Про `.trae/documents/`

У `.trae/documents/` лежать згенеровані Trae-документи (PRD/архітектура/дизайн сторінок). Вони корисні як довідка, але можуть містити кілька версій і не збігатися з поточним кодом.

Рекомендація: підтримувати “канонічну” документацію в `docs/`, а все інше або архівувати, або явно маркувати як чернетки.
