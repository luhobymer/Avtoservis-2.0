# Avtoservis Project Rebuild

## Документація проекту Avtoservis

Цей каталог містить всю документацію проекту.

### Основні файли документації:

- **[DEVELOPMENT_GUIDE.md](DEVELOPMENT_GUIDE.md)** - Повний посібник розробника з налаштування, принципами розробки та командами
- **[DATABASE.md](DATABASE.md)** - Документація по базі даних, моделях та SQL запитах
- **[TESTING_AND_DEBUGGING.md](TESTING_AND_DEBUGGING.md)** - Стратегії тестування, налагодження та Error Boundaries
- **[technical-requirements.md](technical-requirements.md)** - Технічні вимоги до системи
- **[project-analysis.md](project-analysis.md)** - Аналіз проекту та план розвитку

### Допоміжні файли:
 - `models-structure.backup.js` - містить структуру моделей даних
 - `instAPI.txt` - документація API управління Supabase

## Історія проекту

Цей проект є перебудовою системи управління автосервісом з переходом від MySQL до Supabase та створенням мобільного додатка.

### Збережені дані
- Структура моделей даних з попередньої версії
- Налаштування бази даних
- Технічні вимоги та аналіз проекту

## Інструкції для перебудови проекту

1. Використовуйте збережені налаштування бази даних для налаштування нового проекту
2. Створіть нову структуру проекту з нуля
3. Використовуйте збережену структуру моделей даних для створення нових моделей
