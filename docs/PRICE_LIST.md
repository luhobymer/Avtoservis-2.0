# Прайс послуг (ціна/час)

Цей проєкт зберігає базовий прайс у таблиці `services`:

- `price` (грн)
- `duration` (хв)

Для механіків також є можливість задавати оверрайди для конкретного механіка в `mechanic_services`:

- `price_override`
- `duration_override`

## Імпорт прайсу з фото

Скрипт: `server/scripts/seed_price_list.js`

Запуск:

```bash
cd server
node scripts/seed_price_list.js
```

Що робить:

- створює категорії в `service_categories` (якщо їх нема)
- додає/оновлює послуги в `services` за назвою
- заповнює `price` та (де вказано) `duration`

Примітки:

- Діапазони типу "від 250-600" зараз зберігаються як мінімальна ціна (наприклад 250) + текст у `description`, щоб не губити сенс.
- Для погодинних робіт додано окрему послугу "Слюсарні роботи (погодинно)".
