# Avtoservis 2.0

Монорепозиторій:

- `server/` — Backend API (Node.js/Express)
- `client/` — Web клієнт (React/Vite)
- `mobile/` — Mobile застосунок (Expo/React Native)
- `telegram-bot/` — Telegram бот
- База даних — Cloudflare D1 (SQLite)
